# katas
collection of katas

-----
Diamond Kata
http://www.natpryce.com/articles/000807.html


    Given a letter, print a diamond starting with ‘A’ with the supplied letter at the widest point.

    For example: print-diamond ‘C’ prints

      A
     B B
    C   C
     B B
      A
